Main Screen

This is the first screen shown when you startup Irotect. It contains three cards which report the conditions of your mobile phone.

  • Version Card - this card checks for latest irotect version and is red if you does not have latest version.
  • Service Card - this card checks if the irotect service is running and is red if there is some type of interuption in the running of the servces. It is very important for the irotect services to run properly for the app functionality.
  • Root Card - this card checks if your phone has root permissions. Rooted mobiles can perform special commands.

Intruder Selfie Gallery

It us the app's built-in gallery to see the intruder selfies taken by the "#take_intruder_selfie" command. It can be accessed by going to 'main screen->three line menu->intruder selfie'.

Emergency Call

It is one of the key features of irotect. After first opening of the app the app adds a contact "iROTECT SOS" in your phone directory which you can use in time of danger/emergency or you can directly call "252525" from your favourite dialer.

This call triggers a set of tasks in irotect like

  • Turns on vibration mode
  • Increases ringer volme to 100%
  • Sends an SMS to the 2 trusted contacts provided by the user asking them to contact you as soon as possible
  • Within 5 mins sends periodic(per 5 min) sms to both trusted contacts containing the location and other device informations.(these tracking sms can be stopped by using '#stop_tracking' command.)

Settings Screen

This is the main settings screen of the app. It can be accessed in 'menu(three dot menu on top right corner)->settings'.

  • Intruder selfie resolution : helps you to set the resolution of images taken using '#take_intruder_selfie' command.
  • GPS Tracking : Used to trigger the access of '#start_tracking' command. I recomend keeping it ON
  • SMS services : Used to trigger the running of commands through SMS. I recomend keeping it ON
  • Trusted Contacts : these are the contacts who has access to SMS services. Note - Trusted Contact 1 is notified if sim card changes state or if the device is rebooted.

Commands

The key operations in irotect are performed through SMS Commands. The specific features are triggered when the mobile phone recieves a specific command as SMS body which are then processed and presented accordingly. The currently available SMS commands are:

  • #call_me
    This command calls the sender from the device running irotect.
  • #start_tracking
    This command starts the gps tracking and replies with periodic messages with gps location of device and other basic information untill stopped by '#stop_tracking' command.
  • #stop_tracking
    This command stops the periodic tracking of the device started by '#start_tracking' command.
  • #say_loud
    This command speaks the text written after this command inside the SMS body. This command is not guaranted to run in all devices.
  • #emergency_alarm
    This command rings the phone with an emergency siren sound even if the device is silent. This sound can be stopped by '#stop_emergency_alarm' command.
  • #stop_emergency_alarm
    This command is used to stop emergency alarm
  • #take_intruder_selfie
    This command captures a photograph from front and rear camera symultaneously and silently. These images can be accesed from the app by going to (app->three line menu->intruder selfies) or by using '#email_intruder_selfie' command.
  • #email_intruder_selfie
    This command sends an email to the registered email address with the images taken from '#take_intruder_selfie' command as attachment. This command needs internet connection to work properly.
  • #test_service
    This command returns positive response if the Irotect services is running properly. If no SMS is recieved from the device then it is doubtful to say if the services are running or not.
  • #device_info
    This command returns current status of the device and other useful information.
  • #forgot_pattern
    This command returns the hint you saved in the app to remember your keyguard pattern(Note - this command only works if a pattern hint is specified in 'menu->keyguard hint').

Frequently Asked Questions

Why won't Android let me uninstall Irotect ?

It's probably because Irotect is set as an administrator in the android settings. It is used as an security measure to prevent the intruder from uninstalling Irotect from home screen.

You can disable it by going to Android Settings / Location & Security / Device Administration.

Why isn't Irotect tracking my location ?

Please make sure your GPS is turned on and Irotect has permissions to access GPS. If still the app is not able to return your GPS cordinates feel free to contact the Developer.

How can i send the SMS commands ?

This is very easy you just have to send the sms command(in message body one at a time) to the phone number currently activated in the device.
Note -

  • The SMS commands are triggered only if the SMS is sent from one of the trusted contacts saved inside the app.
  • The SMS commands are case sensitive
  • the SMS commands consist of only letters and underscore "_" character

How can I contact the developer ?

There is a contact email given at Menu / About. In general, this is intended for order and validation help, which is often confidential, and other private issues.

Can Irotect be used from an external SD card ?

Prior to Android 6.0, that is not possible, even using apps2SD.

Does Irotect need constant internet connection ?

No, Irotect just needs internet connection on first app start to download basic resources needed for the proper functioning of the app. '#email_intruder_selfie' command is the only command which needs internet connection but just for a couple of seconds.

Can Irotect locate my device if the mobile is switched off ?

It is impossible for any app to run when the device is switched off.

How can i use the Location Coordinates returned by Irotect ?

You can search the coordinates in Google Maps app or website. click here to know more.

Why isn't Irotect availabe in Google Play Store ?

The app is currently under beta development so we have not considered adding it to google play store but we are sure in near future the app will be made available to users in google play store.
